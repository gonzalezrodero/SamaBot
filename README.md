# SamaBot
AI assistant for Club Bàsquet Samà. Engineered by a former QA turned Backend Dev, it features a pro-grade stack: .NET 10, Marten (Event Sourcing), and Wolverine. Using RAG with pgvector on AWS, it delivers local club info with a "Vilanoví" touch. Quality-first code from someone who spent 7 years learning how to break it.
